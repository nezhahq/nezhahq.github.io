# 自建一个TG机器人来查询服务器信息

这是社区项目归档页，当前版本兼容性未验证。使用前请先查看上游仓库状态，并确认其 API 调用方式适用于当前哪吒版本。
:::

机器人可以通过API向面板请求服务器状态信息，得到信息后发送给用户  
你可以搭建此机器人来方便地查看指定服务器的当前状态且不需要打开面板
