# 使用 Cloudflare Access 作为 OAuth2 提供方

::: warning
本文原内容适用于 v0 版本的 OAuth2 配置格式，不适用于 v1 及之后版本。本页仅为保留旧链接而存在，请不要按旧配置格式部署新的 Cloudflare Access 登录。
:::

v1 之后 Dashboard 使用新的 OAuth2 配置结构，Cloudflare Access 也应按新版方式配置。

请直接参考 [设置 OAuth 2.0 绑定](/guide/q14.html#cloudflare-access)。该文档包含当前版本可用的 Callback 地址、配置字段和 Cloudflare Access 示例。
